Atlas

A WebGL projection of points. Points that are more similar in the source embedding tend to land near each other.

Coordinates have no meaning beyond "closer = more similar" — only local distances are reliable.

Navigate

  • ◀▲▼▶ arrow keys — pan
  • ⇧ + ◀▶▲▼ shift+arrows — rotate / tilt (3D only)
  • + / − — zoom in / out
  • scroll wheel — zoom toward cursor
  • left-click drag — orbit (3D) / pan (2D)
  • two-finger touch — orbit on mobile
  • double-click anywhere — fly camera to that point

Interact with documents

  • hover a dot — preview metadata + thumbnail in side panel
  • click a dot — pin the preview panel open
  • click empty space — unpin
  • click a different dot while pinned — switch to that doc
  • "Open document →" button — full doc viewer in new tab

Filter the cloud

Use the legend and color-by controls to explore subsets of the points.

Switch view

  • 2D / 3D button (topbar) — toggle between flat and orbit views
  • Color: dropdown — color points by any attribute
  • edges checkbox — overlay the 2.4M relation-graph edges (heavy)
  • render tuning (bottom-right) — live-adjust dot size, opacity curve, ramp

Positions come from a dimensionality reduction (UMAP / t-SNE / …) of the source embedding.
